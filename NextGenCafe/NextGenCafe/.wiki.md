# Project Summary
The **NextGen Café** is an innovative full-stack café ordering platform that enhances the dining experience through technology. It allows customers to log in via table IDs or QR codes, explore a visually appealing menu with high-quality images, track orders in real-time, and make secure payments. The project streamlines interactions between diners and restaurant staff, providing an efficient and enjoyable ordering process, now featuring augmented reality (AR) experiences for menu items.

# Project Module Description
- **Intro Animation**: Displays a brand animation on launch.
- **Login System**: Users can log in using table IDs or QR codes.
- **QR Code Scanning**: Users can scan QR codes for automatic login.
- **Bottom Navigation**: Modernized navigation for easy access to HOME, MENU, ORDER, and PAY BILL pages, featuring dynamic cart indicators.
- **Menu Page**: Displays a selection of premium dishes with AR integration, cart functionality, and high-resolution images. Prices are now shown in Indian Rupees.
- **Order Page**: Users can track their order status and manage orders.
- **Pay Bill Page**: Displays an itemized bill with enhanced styling, allows users to download a professional PDF receipt, and auto-confirms orders after 3 seconds. Now includes an interactive feedback system.
- **QR Code Generator**: Generates QR codes for table access.
- **Cart Overlay**: Users can view their cart and place orders. Now includes a "Place Order" button and displays prices in Indian Rupees.

- **Backend**: Supabase or Firebase
